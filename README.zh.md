## 📥 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 点击：[🚀 安装 DuoHacker](https://greasyfork.org/en/scripts/551444-duolingo-duohacker)
3. 访问 [Duolingo.com](https://www.duolingo.com)
4. 点击右下角的 **DuoHacker** 按钮
5. 选择 **安全模式**
6. 开始刷取！🎉
